# Controlador_Midi
Controlador Midi desenvolvido para estudos de lógica de programação em Javascript. Trabalhando com arquivos de áudio e aprimorando a lógica dos algorítmos.
Foi utilizado sons percussivos e tem como objetivo compreende melhor as estruturas de código em sites que trabalham com arquitetura de som ao ser utilizado pelos usuários.

Acesse: https://ravenous-tech.github.io/Controlador_Midi/
